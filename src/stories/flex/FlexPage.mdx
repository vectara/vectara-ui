# VuiSpacer

Use VuiFlexContainer and VuiFlexItem to create component layouts. Examples:

- An icon next to a label, centered along the horizontal axis
- A header with a label on the left side and buttons on the right side, centered along the horizontal axis
- An empty state with an image, title, and description, centered along the vertical axis
- An input field next to a button, centered along the horizontal axis
